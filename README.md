Setup Process: 
1. Open a Spreadsheet 
2. Click on Extensions > Apps Script 
3. Copy and paste the from  Code.gs to Code.gs 
4. Change the FOLDER_ID (You find the folder id from the google drive folder URL)
4. Click on the plus icon and create a new file named "Upload.html" and paste the code from "Upload.html"
5. Reload the Spreadsheet
